# AnySlope

AnySlope allows users to read sakamichi's blogs, LINE BLOG, and, Ameba Blog in one application.

![](./ss.png)

## Development

### Setup

```zsh
npm i -g electron@3.0.10 # Install Electron
yarn # Resolve dependencies
```

### Watch & Build

```zsh
yarn watch # Watch mode

yarn build # Build app
```

### Executing App

```zsh
electron .
```
